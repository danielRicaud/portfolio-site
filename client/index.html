Recent Applications

Hackalope


Hackalope is a site of user generated links to useful coding resources and tutorials. You can search, vote, comment, and save resources while filtering by programming language and topic. Hackalope was built using React, Redux, MongoDB, Node.js, and Express. Front-end navigation is handled via React Router and it is styled using Google's Material UI.

CardCast


Cardcast is a collaborative presentation platform that utilizes the Google ChromeCast. Chromecast allows users to create and save slide decks with markdown language. They can project these slides to any device connected to ChromeCast. It was built using AngularJS, MongoDB, Node.js, and Express.

Echoes


Echoes is a living listening history. With Echoes, it's easy to rate,remember, and relive the albums you've enjoyed. This app was inspired by a listening journal that one of our team members has been keeping for the past seven years. It was built on React, PostgreSQL, Node.js, and Express. Echoes was designed using Bootstrap.
